# Credential Certificates

This directory should contain your credentials from TESAIoT Admin UI.

## How to Get Credentials

1. Log in to [TESAIoT Admin UI](https://admin.tesaiot.com)
2. Navigate to **Devices** > Select your device
3. Download QUIC credentials bundle
4. Extract into this directory

## Security Warning

**NEVER commit credentials to version control\!**
