# Applications Examples

Examples for Web and Mobile developers building IoT applications.

## Target Audience

| Developer Type | Primary Stack | Use Case |
|----------------|---------------|----------|
| **Frontend Developer** | React, Vue, Angular | Real-time dashboards |
| **Backend Developer** | Node.js, Python, Go | API integration |
| **Mobile Developer** | React Native, Flutter | Mobile IoT apps |
| **Low-Code Developer** | Node-RED, n8n | Visual automation |

## Categories

```
├── rest-api/           # REST API Integration
│   ├── python-cli      # CLI + API client
│   └── api-gateway     # APISIX patterns
│
├── real-time/          # Real-time Streaming
│   ├── wss-mqtt        # WebSocket MQTT
│   └── live-dashboard  # React dashboard
│
├── visualization/      # Data Visualization
│   ├── react-dashboard # Edge AI dashboard
│   └── grafana         # Grafana dashboards
│
└── visual-automation/  # Low-Code Tools
    ├── nodered         # Node-RED flows
    ├── google-sheets   # Sheets integration
    └── telegram-bot    # Alert bot
```
