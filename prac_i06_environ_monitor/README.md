# Environmental Monitor

## เกี่ยวกับตัวอย่างนี้
จอแสดงผลตรวจวัดสิ่งแวดล้อมแบบเรียลไทม์ แสดงอุณหภูมิ ความชื้น และความดันอากาศ ในการ์ด 3 ใบ พร้อมลูกศรแสดงแนวโน้ม (ขึ้น/ลง/คงที่) และแถบ progress bar ด้านล่างมีตัวประเมินโซนความสบาย (Comfortable, Too Hot, Too Cold, Too Dry, Moderate)

## เซ็นเซอร์ที่ใช้
- **DPS368** — ความดันอากาศ (hPa) และอุณหภูมิสำรอง
- **SHT4x** — อุณหภูมิ (C) และความชื้นสัมพัทธ์ (%)

## สิ่งที่จะได้เรียนรู้
- การสร้าง dashboard card ด้วย `example_card_create()`
- การใช้ `lv_bar` สำหรับ progress indicator
- การคำนวณ trend arrow จากค่าก่อนหน้า
- การประเมิน comfort zone จากหลายตัวแปร
- การจัดเรียง layout ด้วย flex row

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
การ์ด 3 ใบเรียงแนวนอน (Temperature, Humidity, Pressure) แต่ละใบแสดงค่าตัวเลขขนาดใหญ่ ลูกศรแนวโน้ม และ progress bar ด้านล่างสุดมีแถบแสดงสถานะ comfort zone

## ลองปรับแต่ง
- ปรับเกณฑ์ comfort zone ใน `comfort_zone()` ตามสภาพอากาศในพื้นที่
- เพิ่ม heat index calculation จากอุณหภูมิ+ความชื้น
- เพิ่มกราฟ trend history ย้อนหลัง 5 นาที
