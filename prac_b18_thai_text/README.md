# Thai Text Display

## เกี่ยวกับตัวอย่างนี้
ตัวอย่างนี้สาธิตการแสดงผลตัวอักษรภาษาไทยบนหน้าจอ LVGL ด้วยฟอนต์ Noto Sans Thai ในขนาด 14, 16, 20 และ 28 พิกเซล ครอบคลุมกรณีพิเศษของภาษาไทย เช่น สระลอย วรรณยุกต์ซ้อน สระอำ สระหน้า และข้อความผสมไทย-อังกฤษ

## สิ่งที่จะได้เรียนรู้
- วิธีใช้ฟังก์ชัน `thai_label()` สร้างข้อความภาษาไทยบน LVGL
- การเลือกขนาดฟอนต์ที่เหมาะสมกับเนื้อหาต่างๆ
- การจัดการกรณีพิเศษของตัวอักษรไทย (สระลอย, สระอำ, พยัญชนะควบกล้ำ)
- การตั้งสีพื้นหลังและสีข้อความด้วย LVGL

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
หน้าจอพื้นหลังสีน้ำเงินเข้มแสดงข้อความภาษาไทยในขนาดต่างๆ ตั้งแต่ 28px (หัวเรื่อง) จนถึง 14px (รายละเอียด) แต่ละขนาดใช้สีต่างกันเพื่อให้เห็นความแตกต่าง มีเส้นแบ่งคั่นระหว่างหัวเรื่องกับเนื้อหา

## ลองปรับแต่ง
- เปลี่ยนข้อความภาษาไทยเป็นประโยคของตัวเอง ลองใส่สระลอยหรือวรรณยุกต์ซ้อนกัน
- ปรับขนาดฟอนต์ให้ใหญ่ขึ้นหรือเล็กลง (เช่น 12, 24, 32)
- เปลี่ยนสีพื้นหลังและสีข้อความให้เข้ากับธีมของแอป
- เพิ่มข้อความภาษาไทยที่มีตัวเลขไทย (๑๒๓) และเครื่องหมายพิเศษ
- ลองสร้างการ์ดหลายใบที่แสดงข้อความภาษาไทยในหมวดหมู่ต่างๆ
