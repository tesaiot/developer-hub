# Motion Detector

## เกี่ยวกับตัวอย่างนี้
ตัวอย่างนี้สาธิตการตรวจจับการเคลื่อนไหวจากเซ็นเซอร์ BMI270 โดยวิเคราะห์การเปลี่ยนแปลงของค่า magnitude ของความเร่ง เมื่อตรวจพบการเคลื่อนไหวจะแสดง alert สีแดง นับจำนวนเหตุการณ์ และบันทึกลงในบันทึกประวัติแบบ scrollable ใช้ `app_sensor/bmi270` อ่านค่าเซ็นเซอร์โดยตรงผ่าน I2C

## สิ่งที่จะได้เรียนรู้
- การคำนวณ magnitude จากค่าความเร่ง 3 แกน (`sqrtf(ax^2 + ay^2 + az^2)`)
- การทำ motion detection ด้วย threshold และ cooldown
- การใช้ `lv_textarea_create()` เป็น event log แบบ scrollable
- การเปลี่ยนสี indicator ตามสถานะ (เขียว=ปกติ, แดง=ตรวจพบ)
- การจัดการ string buffer เพื่อจำกัดขนาด log

## ความต้องการ
- บอร์ด KIT_PSE84_AI หรือ TESAIoT_PSE84_AI
- เซ็นเซอร์ BMI270 (onboard)
- ไฟล์ `app_sensor/bmi270/` ในโปรเจค

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. ตรวจสอบว่า `app_sensor/bmi270/` อยู่ใน include path
3. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
ด้านบนมีแถบสถานะพร้อมจุดไฟ LED (เขียว/แดง) ข้อความสถานะ จำนวนเหตุการณ์ และค่า magnitude ปัจจุบัน ด้านล่างเป็นบันทึกประวัติเหตุการณ์พร้อม timestamp

## ลองปรับแต่ง
- ปรับ `MOTION_THRESH` เพื่อเปลี่ยนความไวในการตรวจจับ
- ปรับ `COOLDOWN_MS` เพื่อเปลี่ยนระยะเวลา debounce
- เพิ่มเสียงแจ้งเตือนเมื่อตรวจพบการเคลื่อนไหว
- เพิ่มกราฟแสดงค่า magnitude แบบเรียลไทม์
