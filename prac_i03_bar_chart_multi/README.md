# Multi-Series Bar Chart

## เกี่ยวกับตัวอย่างนี้
กราฟแท่งเปรียบเทียบค่าเซ็นเซอร์หลายตัวแบบเรียลไทม์ โดยแปลงค่าจากเซ็นเซอร์ทั้ง 4 ตัว (BMI270, DPS368, SHT4x, BMM350) ให้อยู่ในช่วง 0-100 แล้วแสดงเป็นแท่งกราฟเคียงข้างกัน อัปเดตทุก 500ms

## เซ็นเซอร์ที่ใช้
- **BMI270** — ความเร่ง (Accelerometer) แกน X แปลงเป็น 0-2g -> 0-100
- **DPS368** — ความดันอากาศ 950-1050 hPa -> 0-100
- **SHT4x** — อุณหภูมิ 10-50C -> 0-100 และความชื้น 0-100% โดยตรง
- **BMM350** — ทิศทาง (Heading) 0-360 -> 0-100

## สิ่งที่จะได้เรียนรู้
- การใช้ `lv_chart` ในโหมด `LV_CHART_TYPE_BAR`
- การอ่านค่าเซ็นเซอร์หลายตัวพร้อมกันผ่าน `app_sensor`
- การ normalize ข้อมูลจากหลายแหล่งให้อยู่ในสเกลเดียวกัน
- การอัปเดตกราฟแบบเรียลไทม์ด้วย `lv_timer_create()`

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
กราฟแท่ง 5 แท่ง (Accel, Pressure, Temp, Humidity, Heading) บนพื้นหลังสีเข้ม พร้อมตัวเลขแสดงจำนวนเซ็นเซอร์ที่ทำงานอยู่

## ลองปรับแต่ง
- เปลี่ยนสูตรการ normalize เพื่อปรับช่วงที่แสดงผล
- เพิ่มสีแท่งกราฟแยกตามเซ็นเซอร์แต่ละตัว
- เพิ่ม threshold line เพื่อแสดงค่าเตือน
