# Level Bubble

## เกี่ยวกับตัวอย่างนี้
ตัวอย่างนี้สาธิตระดับน้ำฟองอากาศ (Bubble Level) แบบ 2 มิติ โดยใช้ค่าความเร่งจาก BMI270 แกน X และ Y เพื่อแสดงตำแหน่งฟองอากาศบนตาราง crosshair ทรงกลม เมื่อวางบอร์ดราบ ฟองจะอยู่ตรงกลาง เมื่อเอียง ฟองจะเลื่อนตามทิศทาง พร้อมเปลี่ยนสีตามระดับ ใช้ `app_sensor/bmi270` อ่านค่าเซ็นเซอร์โดยตรงผ่าน I2C

## สิ่งที่จะได้เรียนรู้
- การแปลงค่าความเร่งเป็นตำแหน่ง pixel บนหน้าจอ
- การใช้ `lv_obj_set_pos()` เคลื่อนย้ายวัตถุแบบ dynamic
- การสร้าง crosshair และ inner ring ด้วย LVGL objects
- การเปลี่ยนสี bubble ตามระดับการเอียง (เขียว/เหลือง/แดง)
- การ clamp ค่าตำแหน่งให้อยู่ในขอบเขต

## ความต้องการ
- บอร์ด KIT_PSE84_AI หรือ TESAIoT_PSE84_AI
- เซ็นเซอร์ BMI270 (onboard)
- ไฟล์ `app_sensor/bmi270/` ในโปรเจค

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. ตรวจสอบว่า `app_sensor/bmi270/` อยู่ใน include path
3. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
ด้านซ้ายเป็นวงกลมขนาดใหญ่พร้อม crosshair และวงแหวนด้านใน มีฟองอากาศเคลื่อนที่ตามมุมเอียง ด้านขวาแสดงชื่อ สถานะ (LEVEL/Almost level/Tilted) และค่า X/Y ในหน่วย g

## ลองปรับแต่ง
- ปรับ `SENSITIVITY` เพื่อเปลี่ยนความไวต่อการเอียง
- ปรับ threshold สำหรับสถานะ level/almost/tilted
- เพิ่ม trail effect แสดงเส้นทางที่ฟองเคลื่อนที่
- เพิ่มมุมเอียงเป็นองศาแทนค่า g
