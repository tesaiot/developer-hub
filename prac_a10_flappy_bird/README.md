# Flappy Bird

## เกี่ยวกับตัวอย่างนี้
เกม Flappy Bird ที่ทำงานบนจอทัชสกรีนของ TESAIoT Dev Kit โดยใช้ LVGL สร้าง UI ทั้งหมด สนามเกมขนาด 480x384 พิกเซล มีท่อ 3 ชุดที่เลื่อนเข้ามา นกมีปีกที่ขยับได้ ตา และจะงอยปาก ใช้ธีม Game Boy 4 โทนสีพร้อม CRT scanline overlay เพื่อให้ดูเป็นเกมย้อนยุค

## สิ่งที่จะได้เรียนรู้
- การสร้าง game loop ด้วย `lv_timer_create()` ที่ 50fps (20ms tick)
- ระบบฟิสิกส์แรงโน้มถ่วงและการกระโดด (gravity + flap velocity)
- การจัดการ collision detection ระหว่างนกกับท่อ
- การใช้ touch overlay controls (ปุ่ม Action + ปุ่ม Restart)
- การสร้าง sprite จาก LVGL objects (นก ท่อ)
- ธีม Game Boy 4-tone palette และ CRT scanline effect

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
สนามเกมสีเขียว Game Boy พร้อมนกวงกลมที่ตกลงตามแรงโน้มถ่วง ท่อสีเข้มเลื่อนจากขวาไปซ้าย แตะหน้าจอหรือปุ่ม A เพื่อให้นกกระโดด หลีกเลี่ยงท่อเพื่อทำคะแนนสูงสุด

## ลองปรับแต่ง
- เปลี่ยนค่า `FLAP_PIPE_GAP` เพื่อปรับช่องว่างระหว่างท่อบน-ล่าง (ค่าน้อย = ยากขึ้น)
- ปรับค่าแรงโน้มถ่วง `0.42f` ใน `flappy_step()` เพื่อเปลี่ยนความเร็วการตก
- เปลี่ยนค่า flap velocity `-6.8f` เพื่อปรับแรงกระโดด
- เพิ่มจำนวนท่อด้วย `FLAP_PIPE_COUNT` (ต้องเพิ่ม array size ด้วย)
- เปลี่ยนสีธีมจาก Game Boy เป็นสีอื่นโดยแก้ค่า `GB_DARKEST`..`GB_LIGHTEST`
