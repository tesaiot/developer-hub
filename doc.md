# TESA Developer Hub Curriculum

## Session 1: The Gateway & Data Routing

### Data Ingestion & Gateway
- Routing device telemetry via MQTT
- Utilizing TESAIoT as a centralized hub

### API Configuration & Security
- Setting up REST API/WebSocket (WSS)
- Managing tokens for secure 3rd-party access

### Lab
- Creating API endpoints
- Fetching real-time data using Postman or external scripts

---

## Session 2: The Intelligent 3rd-Parties Platform

### ETL Pipeline Setup
- Preparing and structuring platform data for external application use

### Workflow Automation
- Building automated flows
- Integrating APIs using n8n or Flowwise

### AI Integration
- Introducing TESA X-Brain
- Establishing the RAG architecture

---

## Session 3: Complete Loop Integration & Assessment

### End-to-End Integration
- Connecting the system to user interfaces (e.g., LINE, Web)
- Debugging the full flow

### Knowledge Evaluation
- Conducting the Pre-Post Test to assess technical comprehension

### Project Assessment
- Submitting the Secure AI Project
- Performing a live demonstration for Panel Review
