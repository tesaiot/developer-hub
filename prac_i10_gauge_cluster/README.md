# Gauge Cluster

## เกี่ยวกับตัวอย่างนี้
ตัวอย่างนี้สาธิตการสร้างกลุ่มมาตรวัดแบบ arc gauge 3 ตัว แสดงค่าอุณหภูมิ ความชื้น และความดันบรรยากาศ จากเซ็นเซอร์ DPS368 (ความดัน) และ SHT4x (อุณหภูมิ+ความชื้น) ใช้ `app_sensor` อ่านค่าเซ็นเซอร์โดยตรงผ่าน I2C

## สิ่งที่จะได้เรียนรู้
- การใช้ `lv_arc_create()` สร้าง arc gauge แบบ analog
- การกำหนดช่วงค่า (range) และมุม (bg_angles) ของ arc
- การซ่อน knob ของ arc เพื่อให้เป็น read-only gauge
- การอ่านค่าจากเซ็นเซอร์หลายตัวพร้อมกัน (DPS368 + SHT4x)
- การสร้าง reusable widget function (`make_gauge`)

## ความต้องการ
- บอร์ด KIT_PSE84_AI หรือ TESAIoT_PSE84_AI
- เซ็นเซอร์ DPS368 (onboard) สำหรับความดัน
- เซ็นเซอร์ SHT4x (onboard) สำหรับอุณหภูมิและความชื้น
- ไฟล์ `app_sensor/dps368/` และ `app_sensor/sht4x/` ในโปรเจค

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. ตรวจสอบว่า `app_sensor/dps368/` และ `app_sensor/sht4x/` อยู่ใน include path
3. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
มาตรวัดทรง arc 3 ตัวเรียงกัน: Temperature (-10 ถึง 50 C), Humidity (0-100%), Pressure (950-1050 hPa) แต่ละตัวมีขอบสีต่างกันและแสดงค่าตัวเลขตรงกลาง

## ลองปรับแต่ง
- ปรับช่วงค่าของ gauge ให้เหมาะกับสภาพแวดล้อมที่ใช้งาน
- เพิ่ม gauge ตัวที่ 4 สำหรับค่าอื่น เช่น altitude จาก DPS368
- เพิ่ม animation เมื่อค่าเปลี่ยนแปลง
- เปลี่ยนสี arc indicator ตามระดับค่า (เขียว/เหลือง/แดง)
