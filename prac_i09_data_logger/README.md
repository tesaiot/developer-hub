# Data Logger

## เกี่ยวกับตัวอย่างนี้
ระบบบันทึกข้อมูลเซ็นเซอร์แบบเรียลไทม์ แสดงผลเป็นรายการ scrollable สูงสุด 200 รายการ มีปุ่ม Start/Stop เพื่อควบคุมการบันทึก และปุ่ม Clear เพื่อล้างข้อมูล ข้อมูลแต่ละรายการมี timestamp กำกับ

## เซ็นเซอร์ที่ใช้
- **BMI270** — ความเร่งแกน X (g)
- **DPS368** — ความดันอากาศ (hPa)
- **SHT4x** — อุณหภูมิ (C) และความชื้น (%)

## สิ่งที่จะได้เรียนรู้
- การสร้าง scrollable list ด้วย LVGL
- การจัดการ circular buffer (ลบรายการเก่าสุดเมื่อเต็ม)
- การสร้าง Start/Stop toggle button พร้อม callback
- การอ่านค่าเซ็นเซอร์หลายตัวและ format เป็นข้อความ
- การใช้ `lv_obj_scroll_to_y()` สำหรับ auto-scroll

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
หน้าจอบันทึกข้อมูลพร้อมปุ่ม Stop (สีแดง) และ Clear (สีเทา) รายการข้อมูลจะเพิ่มขึ้นทุกวินาที แสดงค่าเซ็นเซอร์พร้อม timestamp เลื่อนอัตโนมัติไปรายการล่าสุด

## ลองปรับแต่ง
- เปลี่ยน `UPDATE_MS` เพื่อปรับอัตราการบันทึก
- เพิ่ม `MAX_ENTRIES` เพื่อเก็บข้อมูลมากขึ้น
- เพิ่มปุ่ม Export เพื่อส่งข้อมูลผ่าน UART
- เพิ่มเซ็นเซอร์ BMM350 เข้าไปในรายการบันทึก
