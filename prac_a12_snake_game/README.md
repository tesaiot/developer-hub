# Snake Game

## เกี่ยวกับตัวอย่างนี้
เกมงูคลาสสิกที่ทำงานบนจอทัชสกรีนของ TESAIoT Dev Kit ตาราง 30x24 ช่อง (16px ต่อช่อง) งูมีตาและลิ้นที่หมุนตามทิศทาง ควบคุมด้วย Touch D-pad 4 ทิศทาง ใช้ object pool สำหรับลำตัวงูสูงสุด 140 ข้อ พร้อมธีม Game Boy 4 โทนสี

## สิ่งที่จะได้เรียนรู้
- การสร้างระบบ grid-based game ด้วย LVGL
- การใช้ object pool pattern (pre-allocated 140 segments)
- ระบบ edge-triggered input เพื่อป้องกัน 180-degree reversal
- การจัดการ collision detection (กำแพง, ตัวเอง, อาหาร)
- การสร้าง sprite animation (ตาและลิ้นหมุนตามทิศทาง)
- การใช้ Touch D-pad overlay controls

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
สนามเกมสีเขียว Game Boy พร้อมงูที่เคลื่อนที่บนตาราง มีอาหารสี่เหลี่ยมให้กิน ใช้ D-pad ที่มุมซ้ายล่างเพื่อเปลี่ยนทิศทาง เก็บอาหารเพื่อให้งูยาวขึ้นและทำคะแนน

## ลองปรับแต่ง
- เปลี่ยนค่า `SNAKE_TICK_MS` เพื่อปรับความเร็วงู (ค่าน้อย = เร็วขึ้น)
- ปรับขนาดตาราง `SNAKE_GRID_W` / `SNAKE_GRID_H` เพื่อเปลี่ยนขนาดสนาม
- เพิ่มระบบ wrap-around (งูทะลุกำแพงออกอีกฝั่ง) แทน wall collision
- เพิ่มอาหารพิเศษที่ให้คะแนนมากขึ้นหรือเพิ่มความเร็ว
- เปลี่ยนรูปลักษณ์หัวงูโดยปรับตำแหน่งตาและลิ้น
