# RGB Color Mixer

## เกี่ยวกับตัวอย่างนี้
ตัวอย่างนี้สร้างเครื่องมือผสมสี RGB แบบอินเทอร์แอคทีฟ โดยใช้สไลเดอร์สามตัว (แดง, เขียว, น้ำเงิน) ผู้ใช้สามารถลากสไลเดอร์เพื่อผสมสีและดูผลลัพธ์แบบเรียลไทม์ในกล่องพรีวิวขนาดใหญ่พร้อมรหัสสี Hex

## สิ่งที่จะได้เรียนรู้
- การใช้ `lv_slider_create()` สร้างสไลเดอร์แบบกำหนดเอง
- การจัดการ Event Callback เมื่อค่าสไลเดอร์เปลี่ยน
- การคำนวณค่า luminance เพื่อเลือกสีข้อความที่ contrast ดี
- การจัด Layout แบบ Row (ซ้าย-ขวา) ด้วย Flex Flow
- การใช้ `lv_color_make()` สร้างสีจากค่า RGB

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
ด้านซ้ายจะมีการ์ดแสดงสไลเดอร์ 3 ตัว (R, G, B) พร้อมค่าตัวเลข ด้านขวาจะมีกล่องสี่เหลี่ยมขนาดใหญ่แสดงสีที่ผสมได้ พร้อมรหัส Hex ตรงกลาง เมื่อลากสไลเดอร์ สีจะเปลี่ยนแบบเรียลไทม์

## ลองปรับแต่ง
- เพิ่มสไลเดอร์ Alpha (ความโปร่งใส) เป็นตัวที่ 4
- เพิ่มการแสดงค่าสีในรูปแบบ HSL นอกเหนือจาก Hex
- เพิ่มปุ่มสำหรับเซ็ตสีที่ใช้บ่อย (preset colors)
- เปลี่ยนขนาดสไลเดอร์และกล่องพรีวิวให้เหมาะกับหน้าจอ 4.3 นิ้ว
- เพิ่มฟีเจอร์บันทึกสีที่ชอบลง palette
