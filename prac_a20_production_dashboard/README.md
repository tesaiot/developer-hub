# Production Dashboard

## เกี่ยวกับตัวอย่างนี้
แดชบอร์ดการผลิตระดับอุตสาหกรรม แสดงข้อมูลเซ็นเซอร์ 4 ตัวบนการ์ดแยกกัน (BMI270 IMU, DPS368 ความดันอากาศ, SHT4x ความชื้น, BMM350 เข็มทิศ) พร้อมกราฟ Accel Magnitude แบบ rolling chart และแถบสถานะเซ็นเซอร์ด้านล่าง ข้อมูลเซ็นเซอร์อ่านโดยตรงจาก app_sensor โดยไม่ต้องพึ่ง IPC

## สิ่งที่จะได้เรียนรู้
- การเริ่มต้นและอ่านค่าเซ็นเซอร์ทั้ง 4 ตัวผ่าน `app_sensor` reader API
- การสร้างแดชบอร์ดแบบมีหลาย card ด้วย grid layout
- การใช้ `lv_chart_create()` แสดงกราฟ rolling data แบบเรียลไทม์
- การใช้ color-coded status dot แสดงสถานะเซ็นเซอร์
- การแสดง uptime จาก FreeRTOS tick count

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. ตรวจสอบว่า `app_sensor/` อยู่ใน include path
3. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
ด้านบนแสดงชื่อ "A20 - Production Dashboard" พร้อมคำบรรยายภาษาไทย ตรงกลางมีการ์ดเซ็นเซอร์ 4 ใบ (IMU, Pressure, Humidity, Compass) แสดงค่าเรียลไทม์ ด้านล่างมี rolling line chart แสดง accel magnitude และแถบสถานะพร้อม dot indicator สีเขียว/เทาตามสถานะเซ็นเซอร์

## ลองปรับแต่ง
- เพิ่มการ์ดเซ็นเซอร์ตัวที่ 5 สำหรับ gyroscope
- เปลี่ยน chart ให้แสดงข้อมูลเซ็นเซอร์ตัวอื่น (เช่น ความดัน)
- เพิ่ม threshold alert เมื่อค่าเซ็นเซอร์เกินขอบเขต
- ปรับ layout ให้เหมาะกับหน้าจอขนาดต่างๆ
