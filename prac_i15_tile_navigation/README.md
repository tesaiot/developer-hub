# Tile Navigation

## เกี่ยวกับตัวอย่างนี้
ตัวอย่างนี้สาธิตการนำทางแบบไทล์วิว (Tileview) โดยใช้การปัดหน้าจอในแนวนอนเพื่อสลับระหว่าง 4 หน้า ได้แก่ หน้า Sensors, Chart, Status และ About พร้อมตัวบอกตำแหน่งหน้าปัจจุบัน ข้อมูลเซ็นเซอร์ใช้ค่าจำลองเพื่อให้ UI ทำงานได้โดยไม่ต้องพึ่ง IPC

## สิ่งที่จะได้เรียนรู้
- การใช้ `lv_tileview_create()` สร้างหน้าจอแบบปัดซ้าย-ขวา
- การจัดการ Event เมื่อเปลี่ยนหน้า (`LV_EVENT_VALUE_CHANGED`)
- การสร้าง line chart ด้วย `lv_chart_create()` แสดงข้อมูลแบบเรียลไทม์
- การใช้ `lv_timer_create()` อัพเดทข้อมูลเป็นระยะ
- การจัดโครงสร้าง multi-page app ด้วยฟังก์ชันแยกต่างหากสำหรับแต่ละหน้า

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
ด้านบนจะมีข้อความบอกหน้าปัจจุบัน (เช่น "Page 1/4: Sensors") ปัดซ้ายเพื่อไปหน้าถัดไป หน้าแรกแสดงข้อมูล IMU และสภาพแวดล้อม หน้าสองแสดงกราฟ Accel X หน้าสามแสดงสถานะเซ็นเซอร์ หน้าสี่แสดงข้อมูลแอป

## ลองปรับแต่ง
- เพิ่มหน้าที่ 5 สำหรับแสดงข้อมูลเพิ่มเติม
- เปลี่ยนทิศทางการปัดเป็นแนวตั้ง (vertical swipe)
- เพิ่ม dot indicator แทนข้อความบอกหน้า
- ปรับข้อมูลจำลองให้เป็นค่าเซ็นเซอร์จริงผ่าน `app_sensor`
- เพิ่ม animation เมื่อเปลี่ยนหน้า
