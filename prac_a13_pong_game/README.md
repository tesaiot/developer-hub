# Pong Game

## เกี่ยวกับตัวอย่างนี้
เกมปิงปองที่ทำงานบนจอทัชสกรีนของ TESAIoT Dev Kit สนาม 480x384 พิกเซล ไม้ตี 14x90 ลูกบอล 14px ฝั่งขวาเป็น AI คู่ต่อสู้ที่ติดตามลูกบอลด้วยความเร็ว 3.2px/frame ลูกบอลเร็วขึ้นเรื่อยๆ เมื่อกระทบไม้ตี เล่นจนถึง 10 คะแนน

## สิ่งที่จะได้เรียนรู้
- การสร้างระบบฟิสิกส์ลูกบอล (velocity, bounce, deflection angle)
- การเขียน AI ง่ายๆ สำหรับ paddle tracking
- ระบบ progressive difficulty (ลูกบอลเร็วขึ้นทุกครั้งที่กระทบไม้)
- การใช้ continuous hold input สำหรับ smooth paddle movement
- AABB collision detection ระหว่างลูกบอลกับไม้ตี
- การสร้าง UI elements ด้วย LVGL (paddles, ball, center line)

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
สนามปิงปองสีเขียว Game Boy มีเส้นแบ่งกลาง ไม้ตีฝั่งซ้าย (ผู้เล่น) และฝั่งขวา (AI) ลูกบอลกลมเด้งไปมา ใช้ปุ่ม UP/DOWN ที่ด้านซ้ายเพื่อขยับไม้ตี ทำคะแนน 10 ก่อนชนะ

## ลองปรับแต่ง
- ปรับความเร็ว AI ที่ค่า `3.2f` ใน `pong_step()` (ค่ามาก = AI เก่งขึ้น)
- เปลี่ยนขนาดไม้ตี `PONG_PADDLE_H` (ค่าน้อย = ยากขึ้น)
- ปรับความเร็วเริ่มต้นลูกบอล `4.4f` ใน `pong_reset_ball()`
- เพิ่มระบบ 2 ผู้เล่น โดยให้ AI เป็น touch control ฝั่งขวา
- เพิ่ม power-up ที่ทำให้ลูกบอลเร็วขึ้นหรือไม้ตียาวขึ้นชั่วคราว
