# Line Chart Acceleration

## เกี่ยวกับตัวอย่างนี้
ตัวอย่างนี้สาธิตการสร้างกราฟเส้น (Line Chart) แบบเรียลไทม์ โดยแสดงค่าความเร่งจากเซ็นเซอร์ BMI270 ทั้ง 3 แกน (X, Y, Z) ด้วยสีที่แตกต่างกัน มีระบบ autoscale ปรับช่วงแกน Y อัตโนมัติตามข้อมูลจริง ใช้ `app_sensor/bmi270` อ่านค่าเซ็นเซอร์โดยตรงผ่าน I2C

## สิ่งที่จะได้เรียนรู้
- การใช้ `lv_chart_create()` สร้างกราฟเส้นแบบ scrolling
- การเพิ่ม series หลายชุดด้วย `lv_chart_add_series()` พร้อมกำหนดสี
- การทำ autoscale Y-axis แบบ dynamic ด้วย padding 10%
- การอ่านค่า BMI270 ผ่าน `bmi270_reader_poll()` แทน IPC
- การใช้ `lv_timer_create()` อัพเดทข้อมูล 50ms interval

## ความต้องการ
- บอร์ด KIT_PSE84_AI หรือ TESAIoT_PSE84_AI
- เซ็นเซอร์ BMI270 (onboard)
- ไฟล์ `app_sensor/bmi270/` ในโปรเจค

## วิธีติดตั้ง
1. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปที่ `proj_cm55/apps/`
2. ตรวจสอบว่า `app_sensor/bmi270/` อยู่ใน include path
3. Build และ flash ตามปกติ

## สิ่งที่จะเห็นบนหน้าจอ
กราฟเส้น 3 สี (แดง=X, เขียว=Y, ฟ้า=Z) เลื่อนจากขวาไปซ้ายแบบเรียลไทม์ แสดง 100 จุดล่าสุด ด้านล่างมี legend แสดงค่าปัจจุบันของแต่ละแกนในหน่วย g

## ลองปรับแต่ง
- เปลี่ยน `CHART_POINTS` เพื่อแสดงประวัติมากขึ้น/น้อยลง
- ปรับ `UPDATE_MS` เพื่อเปลี่ยนอัตราการอัพเดท
- เพิ่มข้อมูล gyroscope เป็นกราฟชุดที่ 4-6
- เพิ่มปุ่ม pause/resume สำหรับหยุดกราฟชั่วคราว
